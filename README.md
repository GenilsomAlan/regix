# Registro de ponto

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor:
```bash
npm start
```

3. Acesse o app em [http://localhost:3000](http://localhost:3000)

## Funcionalidades
- Registrar ponto (data e hora)
- Ver os últimos 50 registros
- Interface simples em HTML, CSS e JS
